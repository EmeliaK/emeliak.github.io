SHIKAKU_BMSとは、最大4つのキー音を同時に鳴らすことができるという制限形式で開催されたBMSイベント。つまり演奏できるノーツがまだ未配置の状態ではB1、B2、B3、B4のみ使用可能（一般性を失わず）。
SHIKAKU_BMS is a restricted event format where you can only start a maximum of 4 keysounds simultaneously. In other words, while no playable notes are placed yet, only B1, B2, B3 and B4 can be used (without loss of generality).

登録の時に必ずそれぞれの未配置状態ファイル（いわゆる差分用）も同梱してね。誰でも確認できる為（そして差分の為にも）。
Make sure to include the corresponding uncharted files (i.e. for sabuns) when registering. Just so anyone can confirm (and for sabuns too).
